# TypeScript markdown support notes app

A CRUD notes app with markdown support

## Built with

- TypeScript
- React
- Bootstrap


## Features:

- Create, update, read, delete notes
- Notes are saved when page is refreshed
- Markdown support
- Search & filter functionality
- Responsive
